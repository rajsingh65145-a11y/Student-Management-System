# 🎓 Student Management System

A simple and responsive Student Management System built using **HTML**, **CSS**, and **JavaScript**.

This project allows users to manage student records by entering student details and automatically calculating results.

---

## ✨ Features

- ➕ Add Student Name
- 🆔 Add Roll Number
- 📝 Enter Student Marks
- 📊 Automatic Total Marks Calculation
- 📈 Automatic Percentage Calculation
- 🏆 Automatic Grade Generation
- 📋 Display Student Records in a Table
- 🎨 Clean and Responsive User Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 📖 How It Works

1. Enter the student's name.
2. Enter the roll number.
3. Enter the student's marks.
4. Click the **Add Student** button.
5. The system automatically calculates:
   - Total Marks
   - Percentage
   - Grade
6. Student information is displayed in the table.
